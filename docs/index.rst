aioredis
========

asyncio (:pep:`3156`) Redis support.

Features
--------

- Connections pool
- Low-level & high-level API
- :term:`hiredis` parser


Installation
------------

The easiest way to install aioredis is by using the package on PyPi::

   pip install aioredis

Requirements
------------

- Python 3.3 and :term:`asyncio` or Python 3.4+
- :term:`hiredis`
